Linux根目录命令
Created 2020-03-13| Updated 2020-03-11
Post View:

常用命令

  • 查看当前目录 pwd (print work directory)

  • 进入目录 cd+文件夹（change director）

  • cd. 打开当前目录

  • 分为相对路径 和 绝对路径 cd Desktop/aa

    进入aa文件夹 相对路径根据当前所处路径 进入文件夹

    绝对路径 从根目录进入 cd /home/panghu/Desktop/aa

  • 返回上一级目录 cd ..

  • cd - 用于当前目录和上一级目录来回切换

  • 列出文件夹内容 ls

  • 以列表形式列出文件 ls -l

  • 列出隐藏文件夹 ls -a

  • 可以混合使用 ls -l -a

  • 以kb形式显示文件ls -l -h 上面的是以字节显示

  • Tab键可以用来补全单词

  • linux中 想使文件夹隐藏 只需命名以 . 开头

  • 直接回到home目录 cd ~

  • 通配符 * 任意长度字符 （多个字符） ？ 一个长度字符 （一个字符）

    如 * ls 1.dot ** 列出所有以1开头的 .dot文件

  • 通配符 [] 括号的内容 代表一个字符 （一个字符）

    ls [1234] 等同 ls[1-4]

    列出文件名包含1、2、3、4中任意一个字符的文件

  • ls +文件夹 显示指定文件夹内容
